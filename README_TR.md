# Hue

Hue, geleneksel sayısal göstergeler yerine renk spektrumlarını kullanan, minimalist ve odaklanma odaklı bir zamanlayıcı uygulamasıdır. Proje, hem platformlar arası bir mobil uygulamadan hem de bir progresif web uygulamasından (PWA) oluşmaktadır.

## Proje Yapısı

- /hue-mobile: React Native ve Expo ile oluşturulmuş mobil uygulama.
- /hue-web: Next.js, TypeScript ve Framer Motion ile oluşturulmuş web uygulaması.

## Özellikler

- Zaman ayarı ve yönetimi için jest tabanlı kontroller.
- Kalan süreyi yansıtan dinamik renkli arka plan.
- Derin çalışma ve meditasyon için tasarlanmış minimalist kullanıcı arayüzü.
- Masaüstü ve mobil cihazlarda kolay kurulum için PWA desteği.
- Zamanlayıcı tamamlandığında yerel bildirimler.

## Teknolojiler

- React Native (Expo)
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Reanimated

## Başlarken

### Mobil Versiyon
1. hue-mobile dizinine gidin.
2. npm install komutunu çalıştırın.
3. npx expo start komutunu çalıştırın.

### Web Versiyonu
1. hue-web dizinine gidin.
2. npm install komutunu çalıştırın.
3. npm run dev komutunu çalıştırın.
